## 2. PROJECT 

## 2.1 Mini Project 
: boardproject - 미니 프로젝트로 게시판 웹페이지 프로젝트를 진행. 

#### 프로잭트내용
게시판 페이지를 운영하기 위해 게시물 작성, 수정, 삭제 기능과 게시판 목록을 나타내는 기본 기능을 포함하여야 하며 
이후 로그인 기능과 검색 기능을 추가로 생성함. 
게시물의 수가 많아졌을 경우에 대해 페이징 처리기능 추가함.

#### 기능 
- 로그인 
- 게시판 리스트 
- 게시글 작성
- 게시글 수정 
- 게시글 삭제
- 게시글 검색 

#### 구성   
- DAO
  * loginBoard
  * getBoardList
  * getBoardDetail
  * selectBoard
  * insertBoard
  * updateBoard
  * deleteBoard
  * getTotalBoard
  
- VO
  * BoardVO
